Tags de texto

(Voltar)

As tags de texto são tags extremamente importantes no contexto da linguagem de marcação. Em primeiro ponto temos as tags de cabeçalho ou subcabeçalhos. Estas são as famosas tags: h1, h2, h3, h4, h5 e h6. Elas são reponsáveis por definir os títulos (cabeçalhos) e subtítulos das páginas, sendo cada uma de um tamanho diferente de acordo com sua "numeração". Como por exemplo, h1 é comumente usada para títulos e possui um tamanho maior que a h2 que por sua vez costuma ser utilizada para subtítulos. h3, h4.. possuem um tamanho ainda menor e também são utlizadas como subtítulos. Posteriormente temos a tag parágrafo, que é utilizada para textos comuns dentro da página. Além disso, se tem texto na página, em algum momento você irá querer destacar alguns pontos, e por isso existm também as tags de transformar o texto em: itálico, negrito, sublinhado, marcado e etc. São elas: i, strong, u, mark. Para finalizar existe ainda a tag blockquote que é utilizada para fazer uma citação, como por exemplo:

“A tecnologia move o mundo e define o futuro.” Steve Jobs

Tags de Lista

(Voltar)

Existem duas principais tags de lista, são elas: listas ordenadas e listas não ordenadas.. A lista ordenada é aquela que deve seguir uma ordem, uma numeração como 1, 2, 3... e no html é escrita como ol. Por sua vez, uma lista não ordenada não segue nenhum padrão, podem haver itens de qualquer ordem e no html é escrita como ul. É importante ressaltar que esses itens devem ser inseridos nas listas com a tag li, uma vez que é a forma de representar um item.

Tag de Link

(Voltar)

Uma das tags mais utilizadas quando se trabalha com html é a tag de link*, uma vez que ela que permite a navegação dentro de uma página ou de uma página para outra. Representada pela tag a possui um atributo importante que é o href, na qual irá ser inserido para qual local deve ser direcionado dentro de uma página ou para qual página/site você será direcionado ao clicar no elemento ou item em questão. Ao utilizar listas ordenadas ou não ordenadas é comum ver o uso dessa tag dentro da tag li. Esse conteúdo pode ser encontrado na DIO.
